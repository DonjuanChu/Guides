Rails 應用程式模版
===========================

__特別要強調的翻譯名詞__

> Application Template ＝ 應用程式模版。

> Template ＝ 模版。

應用程式模版是一套用來給 Rails 專案添磚加瓦的 DSL。

讀完本篇可能會學到.....

* 怎麼使用模版來產生、客製化 Rails 應用程式。
* 如何用 Rails 模版 API 寫出可重用的應用程式模版。
